Contains Two folder
1.VottingApp - Django application for votting
2.ionicVoting - Ionic project for frontend


Django VottingApp Feature
	
	Contains two types of user
	1.Admin user - can create questions.
	2.Normal user - can create answers.

	Run this command to inside VottingApp Django app to create user

	python manage.py createuser

		enter username,password and role.


ionicVoting features

	Front end for votting

	Uirst login using credentials.

	Admin user - Can create questions.
				 Can View votes for questions.

	Normal User - Can submit their vote for questions.
				  Can View questions and votes voted by them.
